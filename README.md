# ARI

ARI is a simple discord bot made in [discord.js](https://discord.js.org/) / [Typescript](https://typescriptlang.org/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/PastArizox/ARI.git
```

Go to the project directory

```bash
  cd ARI/
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
